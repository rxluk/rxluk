# Lucas Gabriel Costa

## Sobre mim / About Me

---

Sou estudante de Engenharia de Software na UTFPR, atualmente no 7º período, com previsão de conclusão em junho de 2027.

Durante minha formação, construí uma base sólida em desenvolvimento de software, estudando áreas como estruturas de dados, programação orientada a objetos, arquitetura de software, bancos de dados, sistemas operacionais, redes, engenharia de requisitos, qualidade de software e desenvolvimento web.

Foi através da programação orientada a objetos e do ecossistema Java que encontrei minha principal área de interesse: desenvolvimento backend e construção de sistemas robustos.

Tenho experiência prática desenvolvendo APIs REST utilizando Java, Kotlin e Spring Boot, trabalhando com bancos relacionais e não relacionais, Docker, testes automatizados, segurança de aplicações e integração entre sistemas.

Além do backend, possuo conhecimentos em desenvolvimento web, utilizando HTML, CSS e JavaScript, conseguindo atuar em diferentes partes de uma aplicação quando necessário.

No dia a dia, estudo e aplico conceitos relacionados à arquitetura de software, microsserviços, sistemas distribuídos, mensageria, APIs, bancos de dados, infraestrutura Linux e deploy de aplicações. Também mantenho ambientes próprios para testes, servidores e experimentação de novas tecnologias.

Tenho grande interesse por inteligência artificial e computação aplicada. Atualmente estudo fundamentos de Machine Learning, redes neurais, modelos de linguagem, embeddings, sistemas RAG e agentes de IA, buscando entender não apenas como utilizar ferramentas de IA, mas como essas tecnologias funcionam internamente.

Também sou entusiasta da área maker, explorando eletrônica, ESP32, Raspberry Pi, sensores, automação e impressão 3D, unindo software e hardware para criar protótipos e experimentar novas ideias.

Sou curioso, autodidata e gosto de entender como as coisas funcionam por trás dos sistemas. Busco constantemente aprender novas tecnologias e transformar conhecimento em projetos práticos.

Throughout my academic journey, I have built a solid foundation in software development, studying areas such as data structures, object-oriented programming, software architecture, databases, operating systems, networks, software quality, and web development.

Through object-oriented programming and the Java ecosystem, I discovered my main area of interest: backend development and building reliable software systems.

I have practical experience developing REST APIs using Java, Kotlin, and Spring Boot, working with relational and non-relational databases, Docker, automated testing, application security, and system integrations.

Besides backend development, I also have knowledge of web technologies such as HTML, CSS, and JavaScript, allowing me to contribute to different parts of software solutions when needed.

Currently, I study and apply concepts related to software architecture, microservices, distributed systems, messaging, APIs, databases, Linux infrastructure, and application deployment. I also maintain personal environments for testing, servers, and experimenting with new technologies.

I have a strong interest in artificial intelligence and applied computing. I am currently exploring Machine Learning fundamentals, neural networks, language models, embeddings, RAG systems, and AI agents, aiming to understand not only how to use AI tools but also how these technologies work internally.

I am curious, self-taught, and passionate about understanding how things work behind the scenes. I constantly seek new challenges and ways to transform knowledge into practical projects.
